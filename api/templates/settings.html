How weights work

The composite score displayed on the per-page and site health views is a weighted average of all audit type scores.

For example, if SEO has a weight of 20% and scores 80, it contributes 80 × 0.20 = 16 points to the total.

Only audit types that were actually run participate in the calculation — missing audit types are excluded so the weights are re-normalized automatically.

⚠️ Total must be 100%

All 18 weights must sum to exactly 100% (±1% tolerance). The Save button is disabled until the total is in range.
